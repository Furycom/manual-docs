# Knowledge (sources à embedder)

Ce dossier contient des sources documentaires non-canoniques, versionnées localement, destinées à l’indexation/embedding (RAG).
Les fichiers canoniques de démarrage restent dans: /home/furycom/manual-docs/operations (voir CANONICAL_SET_LATEST.md).

Structure:
- knowledge/mcp-manual/exports/ : exports du manuel MCP (md/txt)
- knowledge/mcp-manual/LATEST.md|LATEST.txt : pointeurs vers les exports "latest"

- knowledge/mcp-stack-docs/exports/ : docs MCP stack (ex: API_BRUCE_GATEWAY.md)
- knowledge/mcp-stack-docs/LATEST.md : pointeur vers le doc le plus utile
- knowledge/mcp-manual/extra/ : docs mcp-manual hors exports (ex: backups-and-snapshots.md)
- knowledge/mcp-manual-archive/exports/ : archives de sessions (json/raw.json)
- knowledge/sessions/exports/ : notes de session isolées importées
- knowledge/sessions/LATEST.md : pointeur vers la dernière note utile
